# Contexte temporel du projet

## Année courante

**L’année courante du projet est 2026.**

Tu DOIS utiliser **2026** (et non 2025) pour :

- Les dates dans les documents (ex. « 4 février 2026 », « Date : 2026 »).
- Les mentions « à jour en… », « as of… », « Dernière mise à jour ».
- Les copyrights et mentions de version (ex. « © 2026 », « Dernière mise à jour : 2026 »).
- Toute référence à « aujourd’hui », « cette année », « current year » dans la doc ou les commentaires.

Utilise 2025 (ou une autre année) **uniquement** quand tu fais référence à un événement ou une date passée explicite (ex. « correction appliquée en 2025 »).
